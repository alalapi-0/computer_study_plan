练习 1：先写最小 db.py

把下面这版最小骨架写进去：

import sqlite3
from pathlib import Path

DB_PATH = Path("runs.db")

def init_db():
    conn = sqlite3.connect(DB_PATH)
    cur = conn.cursor()
    cur.execute("""
        CREATE TABLE IF NOT EXISTS runs (
            id INTEGER PRIMARY KEY AUTOINCREMENT,
            input_file TEXT NOT NULL,
            output_file TEXT NOT NULL,
            raw_count INTEGER NOT NULL,
            filtered_count INTEGER NOT NULL,
            final_count INTEGER NOT NULL,
            created_at TEXT NOT NULL
        )
    """)
    conn.commit()
    conn.close()

这里你已经同时练到了三件事：
	•	sqlite3.connect(...)
	•	cursor.execute(...)
	•	CREATE TABLE IF NOT EXISTS ...

这些都直接对应 Python 和 SQLite 的官方文档。

练习 3：给 db.py 加插入函数

继续补：

def save_run(input_file, output_file, raw_count, filtered_count, final_count, created_at):
    conn = sqlite3.connect(DB_PATH)
    cur = conn.cursor()
    cur.execute("""
        INSERT INTO runs (input_file, output_file, raw_count, filtered_count, final_count, created_at)
        VALUES (?, ?, ?, ?, ?, ?)
    """, (input_file, output_file, raw_count, filtered_count, final_count, created_at))
    conn.commit()
    conn.close()

这里的重点是用 ? 占位符做参数绑定，而不是拼接字符串。Python sqlite3 官方文档把参数绑定作为标准用法；SQLite INSERT 文档也说明 INSERT INTO ... VALUES (...) 是基础形式。
