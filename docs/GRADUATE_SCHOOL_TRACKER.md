# Graduate School Tracker · 院校信息跟踪表（v1.0 · 仅模板）

> **本文档是模板与填写规则，不含任何具体院校的真实招生数据。**
> 任何院校的专业代码 / 考试科目 / 复试要求 / 录取信息，**必须以目标学校研究生院或学院官网当年招生目录为准**。
> 严禁基于经验贴、过期年份、社交媒体截图作为最终依据。

---

## 1. 文档用途

本文档帮助你建立一个**可滚动更新**的 0854 / 计算机相关考研目标院校信息库。

主要服务于：

- Stage 6（目标院校与方向筛选）
- 决定数学二 / 408 是否投入 + 投入多少
- 决定是否调整长期目标（例如某些院校只考软件工程综合，则 408 优先级降低）

---

## 2. 严禁事项

- ❌ 不允许基于"我记得 XX 大学考……"填写
- ❌ 不允许基于"去年 XX 学校招……"填写
- ❌ 不允许复制经验贴中的数据
- ❌ 不允许把社交媒体截图作为最终依据
- ❌ 不允许在没有"数据来源链接 + 日期"的情况下提交一行院校数据

---

## 3. 数据来源原则

| 来源等级 | 类型 | 是否可用作最终依据 |
|---|---|---|
| 一级（最权威） | 目标院校研究生院 / 学院官网当年招生目录、专业目录、考试大纲 | ✅ |
| 二级（辅助） | 中国研究生招生信息网（yz.chsi.com.cn）的统招目录 | ✅（与一级交叉验证后） |
| 三级（仅参考） | 教育部学位与研究生教育发展中心、考试报名公告 | ✅（仅用于报名时间等行政信息） |
| 四级（**禁用作最终依据**） | 经验贴、知乎、小红书、贴吧、培训机构资料 | ❌ |

> **任何来自四级的内容**可以记录在 `备注` 列，但**不允许写进 `初试科目 / 是否数学二 / 是否408 / 复试内容` 等关键字段**。

### 4.1 必填字段说明

| 字段 | 必填 | 说明 |
|---|---|---|
| 学校 | 是 | 全名（"清华大学"而不是"清华"） |
| 学院 | 是 | 二级学院 / 系名（精确到招生单位） |
| 专业代码 | 是 | 6 位数字（例：085410） |
| 专业名称 | 是 | 与官方目录一致 |
| 学习方式 | 是 | 全日制 / 非全日制（部分院校两者考题不同） |
| 初试科目 | 是 | 4 门一组（含科目代码 + 科目名） |
| 是否数学二 | 是 | 必须明确 |
| 是否408 | 是 | 必须明确 |
| 复试内容 | 是 | 笔试 / 机试 / 综合面试 |
| 数据来源链接 | **必填** | 院校官网链接（精确到招生目录页面） |
| 数据时点 | **必填** | 该数据来源的发布或采集日期（YYYY-MM） |

### 4.2 选填字段说明

| 字段 | 说明 |
|---|---|
| 是否适合跨考 | 是否限制本科专业 |
| 学校知名度 | 985 / 211 / 双一流 / 其他 |
| 难度自评 | 基于报录比、初试分数线、复试通过率综合 |
| 备注 | 学费、城市、推免比例、招生人数、是否有专项计划等 |

## 5. 填写工作流

> 每填一所学校，按以下步骤走一遍：

1. **打开目标院校研究生院 / 学院官网**，找到当年招生目录。
2. **复制专业代码与专业名称**到主表。
3. **抄写初试科目**，确认是否数学二 / 是否 408。
4. **抄写复试内容**，关注是否需要机试 / 编程语言 / 项目展示。
5. **记录数据来源链接 + 采集日期**。
6. **可选：在备注列写自己的观察**（学费、城市、是否限制本科专业、招生人数）。
7. 一年一更新，建议每年 9–10 月（招生目录发布期）滚动更新。

## 6. 院校筛选优先级建议

仓库不会自动判定院校好坏，但建议按以下"自检维度"打勾：

| 维度 | 你的回答 |
|---|---|
| 1. 数学二 / 408 / 软件工程综合 一致性 | 该校考的是否与你长期投入方向一致？ |
| 2. 是否限制本科专业 | 跨考是否会被刷？ |
| 3. 报录比 / 分数线（多年） | 难度是否在你能承受范围？ |
| 4. 复试机试要求 | 你的代码能力是否需要补 C++ / Java？ |
| 5. 城市 | 学习成本 + 未来工作机会 |
| 6. 学费 | 全日制 / 非全日制学费差异（部分专硕学费较高） |
| 7. 调剂友好度 | 是否能调剂其他院校？ |

## 7. 多专业目标（0854 子方向）

如果你同时考虑多个 0854 子方向（人工智能 / 计算机技术 / 软件工程），可在主表多行记录，但用**同一所学校**的不同行表示：

| 学校 | 专业代码 | 专业名称 | ... |
|---|---|---|---|
| 某校 | 085410 | 人工智能 | ... |
| 某校 | 085404 | 计算机技术 | ... |
| 某校 | 085405 | 软件工程 | ... |

这样后续筛选时可以横向对比同一所学校内的子方向差异。
