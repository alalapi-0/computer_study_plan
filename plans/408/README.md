## 1. 重要声明

- 本目录**不缓存**任何具体 408 考点、真题或解题方法。
- 具体考试范围、题型、教材必须以**最新考研 408 大纲**为准（中国研究生招生信息网 / 教育部考试中心发布）。
- 本目录承载的是"模块清单 + 与软考差异 + 长期节奏"。

408 计算机学科专业基础综合 4 大模块：

| 模块代码 | 模块名 | 与软考差异 | 笔记骨架文件 |
|---|---|---|---|
| `ds` | 数据结构（含算法） | 软考是概念 + 简单代码；408 是综合应用大题 + 手写算法 | [ds.md](ds.md) |
| `composition` | 计算机组成原理 | 软考数制 / 指令系统浅；408 完整 CPU / cache / 流水线计算题 | [composition.md](composition.md) |

| `network` | 计算机网络 | 软考协议功能；408 滑动窗口 / 子网 / 拥塞控制计算题 | [network.md](network.md) |

> 4 份笔记已有启动骨架；是否进入 408 强训练仍按 Stage 5 启动条件决定。

## 3. 与软考的关键关系

- 软考是 408 的**广度版**，不是**替代品**
- 软考通过 ≠ 408 通过
- 软考章节笔记 (`plans/soft_exam/<module>.md`) 与 408 章节笔记 (`plans/408/<module>.md`) **共享概念部分**，**独立加深部分单独写**
- 错题分两份归档：`records/error_notes/soft_exam/<module>/` 与 `records/error_notes/cs408/<module>/`

---

## 4. 推进顺序建议

1. 启动条件：已在 `plans/soft_exam/` 完成至少 5 个模块的概念地图
2. 顺序：
   - 先 **数据结构**（与 0854 / 软考都最重叠，回报最大）
   - 再 **操作系统**
   - 然后 **计算机网络**
   - 最后 **计算机组成**（独立性最强，启动也最累）

- 408 投入与软考真题强度**反向调节**：软考考前 6 周不动 408，软考考完 1–2 周内启动 408
