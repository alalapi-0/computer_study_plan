## 1. 建议章节清单（骨架，非官方考点表）

按主流软件设计师大纲结构整理，学习时以官方当年大纲核对章节取舍。

| 序号 | 章节主题 | 学习要点（概念层） |
|---|---|---|
| 1 | 绪论与基本概念 | 数据、数据元素、数据项；逻辑结构 vs 物理结构 |
| 2 | 线性表 | 顺序表、链表；基本操作与时间复杂度直觉 |
| 3 | 栈与队列 | LIFO / FIFO；典型应用（表达式、层次遍历入口） |
| 4 | 串 | 模式匹配概念；KMP 思想（软考偏理解，408 偏推导） |
| 5 | 数组与特殊矩阵 | 压缩存储；对称 / 三角矩阵地址计算 |
| 6 | 树与二叉树 | 遍历、性质、线索二叉树概念 |
| 7 | 图 | 存储、遍历、最小生成树 / 最短路径概念 |
| 8 | 查找 | 顺序 / 折半 / 散列；B 树 / B+ 树概念 |
| 9 | 排序 | 常见排序分类、稳定性、复杂度对比 |
| 10 | 算法分析与设计 | 时间 / 空间复杂度；分治、贪心、动态规划入门 |

> 每章笔记建议结构：**定义 → 关键性质 → 易混点 → 与 408 差异**（见 `plans/soft_exam/README.md` §5）。

## 2. 与 408 的差异说明

| 维度 | 软考（本笔记） | 408 |
|---|---|---|
| 深度 | 广度 + 概念 + 简单代码阅读 | 综合应用题、手写算法与推导 |
| 题型倾向 | 选择题、下午题代码补全 / 读程序 | 大题、计算与证明 |
| 算法 | 理解常见思想与复杂度 | 常要求完整过程与边界分析 |
| 笔记策略 | 概念清晰、易混点表 | 同模块另建 `plans/408/ds.md`，错题分 lane 归档 |

共享原则：概念可互相引用，**错题不混写**（`records/error_notes/soft_exam/ds/` vs `records/error_notes/cs408/ds/`）。
