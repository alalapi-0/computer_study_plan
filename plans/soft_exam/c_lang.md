## 2. 建议章节清单（启动版）

1. 基本类型、表达式与控制流
2. 数组、字符串与指针基础
3. 函数、参数传递与作用域
4. 结构体、链表与基本数据结构表达
5. 文件和标准输入输出基础
6. 常见错误：越界、空指针、未初始化、类型转换

---

## 3. 与 408 / 工程实操的关系

- 408 数据结构通常需要用 C/C++ 风格表达算法，C 语言要能支撑链表、树、图的结构写法。
- Python 工程线可以提供算法直觉，但不能替代 C 的指针和内存模型。

## 4. 首轮推进建议（1 周）

- 第 1-2 天：控制流 + 函数
- 第 3-4 天：数组、字符串、指针
- 第 5 天：结构体和链表最小例子
- 第 6-7 天：整理 3 个易错代码片段的错因
