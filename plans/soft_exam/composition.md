## 2. 建议章节清单（启动版）

1. 计算机系统概述（硬件 / 软件划分、冯·诺依曼结构）
2. 数据的表示与运算（进制转换、原码/反码/补码、浮点数概念）
3. 存储系统（内存层次、Cache 概念、虚拟存储基本思想）
4. 指令系统（指令格式、寻址方式、操作类型概念）
5. 中央处理器（运算器/控制器、流水线基本概念、中断概念）
6. 总线（总线类型、带宽与仲裁概念）
7. 输入输出系统（I/O 方式：程序查询 / 中断 / DMA 概念对比）
8. 可靠性基础（校验码、冗余、RAID 概念）

---

## 3. 与 408 的差异提示

- 软考计组为**概念广度 + 基础换算**；408 计组要求**完整 CPU/Cache/流水线计算题与推导**。
- 软考常考进制转换、补码表示、Cache 命中率概念；408 会深入替换算法与性能计算。
- 与操作系统模块（`os.md`）在"内存管理 / 中断"处有交叉，建议两模块交替推进并标注共享概念。

---

## 4. 首轮推进建议（1 周）

- 第 1-2 天：数据表示与进制/补码
- 第 3-4 天：存储层次 + Cache 概念
- 第 5 天：指令系统 + CPU 基本概念
- 第 6-7 天：I/O 方式对比 + 与网络模块薄弱点回看

---

## 5. 易错提醒（启动级）

- 容易在补码运算题中忘记符号位参与运算规则；做题前先写出"真值 → 补码"再运算，避免直接心算。
