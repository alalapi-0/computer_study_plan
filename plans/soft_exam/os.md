## 1. 建议章节清单（骨架，非官方考点表）

| 序号 | 章节主题 | 学习要点（概念层） |
|---|---|---|
| 1 | 操作系统概述 | 功能、特征、运行环境；批处理 / 分时 / 实时概念 |
| 2 | 进程管理 | 进程与线程；状态转换；调度概念（FCFS、SJF、优先级等） |
| 3 | 进程同步与通信 | 临界区、互斥、信号量；PV 操作（软考常考理解型） |
| 4 | 死锁 | 必要条件、预防 / 避免 / 检测概念 |
| 5 | 存储管理 | 分页、分段、段页式；虚拟存储概念 |
| 6 | 文件管理 | 文件逻辑结构、目录结构、存取方式 |
| 7 | 设备管理 | I/O 控制方式；缓冲、SPOOLing 概念 |
| 8 | 作业与用户接口 | 作业管理、Shell 与系统调用概念（可与 Linux 实操对照） |

> 软考侧强调**概念广度**与典型场景；408 侧常要求**调度与内存的数值计算**，见下节差异。

## 2. 与 408 的差异说明

| 维度 | 软考（本笔记） | 408 |
|---|---|---|
| 深度 | 概念 + 简单 PV / 调度理解 | 调度算法计算、银行家算法、页面置换计算 |
| 与 Linux 实操 | Round 00/01 命令行经验可作「使用层」对照 | 408 不考命令，考原理与计算 |
| 笔记策略 | 易混概念表 + 典型小题场景 | 独立 `plans/408/os.md`，错题分 lane |

共享原则：进程 / 内存 / 文件等**名词与框图可共用**，计算题与错题**分开归档**。
