# 软考中级（软件设计师）· UML / 设计模式模块骨架

> ⚠ 注意：最新考试大纲、题型与范围以官方发布为准：<https://www.ruankao.org.cn/>
> lane：`soft_exam`
> 关联 408：408 不单独考 UML；本模块为软考独有

---

## 1. 学习边界说明

- 本文件仅做学习骨架，不缓存具体考题和官方考点原文。
- 章节安排用于推进节奏，不等同于官方考纲条目。

---

## 2. 建议章节清单（启动版）

1. UML 概述与图分类（结构图 vs 行为图）
2. 类图（类、接口、关联、聚合、组合、泛化、依赖）
3. 用例图（参与者、用例、关系）
4. 顺序图 / 通信图（对象交互与消息传递）
5. 活动图 / 状态图（流程与状态变迁）
6. 组件图 / 部署图（系统结构与物理部署概念）
7. 设计模式入门（创建型 / 结构型 / 行为型分类与代表模式）
8. 常见模式识别（单例、工厂、观察者、策略、装饰器等概念与应用场景）

---

## 3. 与 408 的差异提示

- UML 与设计模式为**软考独有**，408 不直接考查图题或模式名称。
- 软考常考"读图识关系"与"根据场景选模式"，不要求手写完整 UML 工具输出。
- 面向对象模块（`oo.md`）中的类与继承概念与本模块交叉，建议两模块交替推进。

---

## 4. 首轮推进建议（1 周）

- 第 1-2 天：类图 + 用例图
- 第 3-4 天：顺序图 + 活动图 / 状态图
- 第 5 天：设计模式分类 + 常见模式识别
- 第 6-7 天：综合读图练习 + 与软件工程模块交叉复习

---

## 5. 易错提醒（启动级）

- 容易混淆类图中聚合（空心菱形）与组合（实心菱形）的语义差异；读图时先判断生命周期是否绑定。
