## 1. 学习边界说明

- 本文件仅做学习骨架，不缓存具体考题和官方考点原文。

---

## 2. 建议章节清单（启动版）

1. 网络体系结构（OSI / TCP-IP 分层、各层职责与典型协议）
2. 物理层与数据链路层（传输介质、帧、MAC、差错控制概念）
3. 网络层（IP 地址、子网划分、路由选择、ICMP 概念）
4. 传输层（TCP / UDP 特性、端口、连接管理、可靠传输概念）
5. 应用层（HTTP / DNS / FTP / SMTP 等常见协议功能）
6. 网络安全基础（加密、认证、防火墙、常见攻击类型概念）
7. 无线与移动网络（WLAN、蜂窝网络基本概念）
8. 网络设备与拓扑（交换机、路由器、网关、典型组网）

---

## 3. 与 408 的差异提示

- 软考侧重**协议功能、分层识别、概念对比**；408 侧重**滑动窗口计算、子网划分数值题、拥塞控制推导**。
- 软考不要求手写复杂网络计算，但需能识别典型场景（如 TCP 三次握手、子网掩码含义）。
- 工程实操线（Round 06、`plans/linux/networking.md`）中的 `ping` / `curl` / `ss` 命令可作为理解辅助，答题仍以软考标准表述为准。

---

## 4. 首轮推进建议（1 周）

- 第 1-2 天：体系结构 + 物理/链路层
- 第 3-4 天：网络层 + 传输层（TCP/UDP 对比）
- 第 5 天：应用层协议 + 安全基础
- 第 6-7 天：与计算机组成模块交叉复习 + 错题整理

---

## 5. 易错提醒（启动级）

- 容易混淆 TCP 与 UDP 的适用场景（可靠有序 vs 低延迟无连接）；做题前先判断题目强调的是"可靠性"还是"实时性"。
