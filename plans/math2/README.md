## 4. 推进顺序建议

```
M1     启动：极限 + 矩阵入门
M2     一元微分
M3     一元积分
M4     极限/微分/积分 综合滚动
M5     线性代数：向量 + 线性方程组
M6     微分方程
M7     多元微分 / 二重积分（按大纲）
M8     特征值 + 二次型
M9+    真题阶段
```

> M = month。**这是粗略节奏**，实际进度取决于你的精力分配。
