## 本轮学会的 7 个命令

```
pwd     看我现在在哪
ls      看这里有什么
cd      去别的地方
mkdir   创建目录（文件夹）
touch   创建一个空文件
cat     把文件内容显示出来
man     查某个命令的说明书
```

---

## 常用路径写法速查

```
~           home 目录（/Users/你的用户名）
.           当前目录
..          上一级目录
~/xxx       绝对路径（从 home 出发）
../xxx      相对路径（从当前位置出发，先回上级）
```

---

## 常用组合

```bash
# 看当前目录 + 内容
pwd && ls

# 创建多级目录（一步到位）
mkdir -p 一级/二级/三级

# 进入目录 + 看内容
cd ~/cli-lab/round0 && ls

# 写内容到文件（覆盖）
echo "内容" > 文件名.txt

# 追加内容到文件（不覆盖）
echo "内容" >> 文件名.txt
```
