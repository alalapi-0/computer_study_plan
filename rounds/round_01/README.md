1. 每周先阅读 `weekN/notes.md`，再执行 `weekN/exercises.sh`。
2. 练习默认在 `~/cli-lab/round1` 沙盒执行，避免误操作仓库文件。
3. 本轮为最小骨架版，打卡任务将在后续轮次接入进度系统。
