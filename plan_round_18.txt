先准备第二份练习数据

进入目录：

cd ~/cli-lab/round18/week3

创建一个更像“标注结果分析”的 CSV：

这里故意放了一个缺失值，方便你练“看缺失”。

练习 12：看缺失值和基本结构

新建：

touch groupby_lab.py

写入：

import pandas as pd

df = pd.read_csv("runs.csv")

print(df.head())
print(df.shape)
print(df.isna().sum())

groupby 参考页里同时能看到 isna()、dropna() 等相关 DataFrame 操作；这也说明缺失值检查本来就是表格处理的第一层动作。 ￼

练习 13：按 label 分组计数

继续补：

import pandas as pd

df = pd.read_csv("runs.csv")

print(df.groupby("label").size())

这一步是在练“不同类别分别有多少条”。groupby 官方文档说明，它就是用来按列分组并对这些组做操作的。 ￼

练习 14：按 source 分组看平均分

继续补：

import pandas as pd

df = pd.read_csv("runs.csv")

print(df.groupby("source")["score"].mean())

pandas 统计教程展示了按类别做均值这种非常典型的分析方式。 ￼

练习 15：组合条件做最小分析

继续补：

import pandas as pd

df = pd.read_csv("runs.csv")

filtered = df[df["score"] >= 0.5]
print(filtered.groupby("label").size())
print(filtered.groupby("source")["length"].mean())

这一题是在练“先筛、再分组、再聚合”，这是后面做数据集分析时非常常见的一条小链路。

练习 16：导出更干净的数据

继续补：

import pandas as pd

df = pd.read_csv("runs.csv")

clean = df.dropna(subset=["score"])
clean = clean[clean["score"] >= 0.3]
clean.to_csv("runs_clean.csv", index=False)
print(clean)

这里你练到了：
	•	缺失值过滤
	•	阈值过滤
	•	导出清洗后结果

这就是最小的数据分析前置流程。

练习 17：把结果写成小结

新建一个文件：

touch analysis_notes.md

用自己的话写 4 句话：
	•	这份数据一共有多少行
	•	缺失值在哪里
	•	哪个 label 最多
	•	哪个 source 的平均 score 更高

这一题看起来简单，但它很重要，因为它在训练你把“代码结果”翻译成“分析结论”。
